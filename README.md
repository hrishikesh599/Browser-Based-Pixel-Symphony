# Browser-Based Pixel Symphony

**Browser-Based Pixel Symphony** is an interactive visual art project inspired by an idea from the Hack Club Idea Box. The original challenge was to create swirling lines that react to mouse movement and dynamically follow the user's cursor.

During development, the project evolved significantly after discovering **tubes1.min.js**, which enabled smoother and more visually appealing motion effects. What began as a simple mouse-following line experiment transformed into an immersive browser-based visual experience that feels alive and responsive.

---

## Features

* Interactive mouse-responsive animations
* Dynamic flowing tube-like visual effects
* Smooth motion and particle behavior
* Lightweight browser-based implementation
* Real-time visual feedback based on cursor movement
* Browser-based visual art experience

## Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/Browser-Based-Pixel-Symphony.git
cd Browser-Based-Pixel-Symphony
```

### Create a Virtual Environment (Optional)

```bash
python -m venv venv
```

Activate it:

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Running the Project

Start the Flask application:

```bash
python app.py
```

The local server will start and the project will be available at:

```text
http://127.0.0.1:5000
```

Open the URL in your browser and move your mouse around the screen to interact with the visual effects.
