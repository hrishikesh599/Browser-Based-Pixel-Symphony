# Browser-Based-Pixel-Symphony
Originally an idea from the idea box of macando,hackclub
i was told to try and make a swirly lines that react with mouse and move accordingly, and for this project i started making lines follow the mouse as the project goes i came across [tubes1.min.js], which changed the whole project into next level, and was perect for this project, 
this project contain:
 * index.html
 * tubes.min.js
 * app.py

 how to run this ?
   * first add the app.py into a folder (eg: program)
   * then create two folders
      1 templates
      2 static
   * add index.html to templates and tubes1.min.js to static folder (already in format)
   * run the app.py 
   * it will host a local server on ip 127.0.0.1:5000
   * use a browser to enter the page

   files structure :
  program
     |
     [templates] - index.html
     |
     [static] - tubes1.min.js
     |
     app.py
